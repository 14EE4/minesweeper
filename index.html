Minesweeper — 파이썬 버전

이 저장소의 간단한 소개 페이지입니다. 이 게임은 로컬에서 Python(데스크톱)으로 실행되는 터미널 / Tkinter 기반 프로그램입니다.

실행 방법 (로컬)

이 프로젝트는 브라우저에서 실행되지 않습니다. 로컬에서 실행하려면 Python 3와 tkinter가 필요합니다.

  1. 저장소를 클론하거나 파일을 복사합니다.
  2. 터미널에서 아래와 같이 실행합니다:
python3 minesweeper5.3.py
# 또는
python3 minesweeper_gui.py

만약 ModuleNotFoundError: No module named 'tkinter' 오류가 발생하면 시스템에 Tk가 설치되어 있지 않은 것입니다. 관리자 권한이 없을 때는 Miniconda를 사용해 로컬에 설치하는 방법이 편합니다.

tkinter 설치 (sudo 권한 없을 때 추천)

Miniconda를 사용하면 시스템을 변경하지 않고 로컬에 파이썬 환경을 만들 수 있습니다.

# Miniconda 설치 (다운로드 후 설치)
curl -LO https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh -b -p $HOME/miniconda3
export PATH="$HOME/miniconda3/bin:$PATH"
conda create -n minesweeper python=3.11 tk -c conda-forge -y
conda activate minesweeper
python minesweeper5.3.py
